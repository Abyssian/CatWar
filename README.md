# CatWar
Скрипты для Tampermonkey
1) CatWar Energy Percent (Overlay)-1.0.user.js
Скрипт отображает процент энергии кота в боевом режиме, фиксируя его в нижней части модели персонажа.

Основан на скрипте  
https://openuserjs.org/scripts/yurei72/CatWar_Energy_Percent/source

В данной версии:
- изменён способ позиционирования процентов энергии;
- проценты отображаются в стабильном положении относительно `.catWithArrow`;
- используется overlay-подход, чтобы избежать влияния верстки и CSS CatWar;
- текст не вращается вместе со стрелкой и не влияет на размеры контейнера.
